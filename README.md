# TermBot

A simple python script that can:

### Math:

- Calculate
- Convert weight

### Space:

- Live ISS Tracker
- Astronomy Picture Of the Day

### Weather:

- Tell the weather

### Fun:

- Rock paper Scissors
- Scrambler/Timer
- System Info
- Countdown

##### And most importantly:

- Exit

### API keys required:

- [OpenWeatherMap](https://openweathermap.org/api)
- [NASA_Api](https://api.nasa.gov/)
